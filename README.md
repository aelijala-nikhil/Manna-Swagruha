# 🍬 Manna Swagruha Website

A modern and responsive landing page designed for **Manna Swagruha**, a traditional homemade sweets and snacks brand.  
The website showcases authentic Indian sweets, services, customer testimonials, gallery, and contact information with a smooth and attractive UI.

---

## 🌟 Features

- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🍬 Menu section with sweets and snacks
- ⭐ Dish ratings with price and offer pricing
- 📦 Services section for packaging and delivery
- 🖼️ Image gallery showcase
- 💬 Customer testimonials slider
- 📍 Google Maps location integration
- 📞 Contact section with form
- 💚 WhatsApp floating button for quick orders
- ✨ Smooth animations and hover effects

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome Icons**
- **Swiper.js** (for testimonials slider)
